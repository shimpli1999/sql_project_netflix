# sql_project_netflix
Netflix Data Analysis Using SQL
=

Overview:
=
This project uses SQL to analyze the Netflix Movies and TV Shows dataset and answer business-driven questions related to content distribution, ratings, geography, genres, actors, and release trends. The goal is to demonstrate strong SQL querying and analytical thinking through realistic use cases.

Dataset: Public Netflix Movies & TV Shows dataset (CSV)
=
Key columns: type, title, director, cast, country, release_year, date_added, rating, duration, listed_in, description
=
Business Questions Covered:
=
Movies vs TV Shows count

Most common ratings by content type

Year-wise and country-wise content trends

Top-producing countries and actors

Longest movie and multi-season TV shows

Genre distribution and documentary analysis

Director- and actor-based filtering

Content classification using keywords (kill, violence)

(15 business questions solved in total)

Skills Demonstrated:
=
SQL Aggregations (COUNT, AVG, GROUP BY)

Window Functions (RANK, ROW_NUMBER)

Date filtering and time-based analysis

String and text analysis

Conditional logic (CASE WHEN)

Data quality checks (NULL handling)
